## Installation

HNG React is built with Create React App and React 18. Follow these steps to get started quickly.

<Steps>
  <Step title="Clone and Install Dependencies">
    First, clone the repository and install the required dependencies:

    <CodeGroup>
    ```bash npm
    npm install
    ```

    ```bash yarn
    yarn install
    ```

    ```bash pnpm
    pnpm install
    ```
    </CodeGroup>

    This will install all dependencies including:
    - React 18.2.0
    - React DOM 18.2.0
    - React Scripts 5.0.1
    - Testing libraries (Jest DOM, React Testing Library)

    <Note>
      The installation may take a few minutes depending on your internet connection.
    </Note>
  </Step>

  <Step title="Start the Development Server">
    Launch the development server to see your app in action:

    <CodeGroup>
    ```bash npm
    npm start
    ```

    ```bash yarn
    yarn start
    ```

    ```bash pnpm
    pnpm start
    ```
    </CodeGroup>

    You should see output similar to:

    ```bash
    Compiled successfully!

    You can now view hng in the browser.

      Local:            http://localhost:3000
      On Your Network:  http://192.168.1.x:3000

    Note that the development build is not optimized.
    To create a production build, use npm run build.
    ```

    The app will automatically open in your browser at [http://localhost:3000](http://localhost:3000).
  </Step>

  <Step title="Verify the App is Running">
    You should see the default React app with:
    - A spinning React logo
    - The text "Edit src/App.js and save to reload."
    - A "Learn React" link

    The page will automatically reload when you make changes to the source code.
  </Step>

  <Step title="Make Your First Code Change">
    Let's modify the app to verify everything is working. Open `src/App.js` and update the component:

    ```jsx src/App.js
    import logo from './logo.svg';
    import './App.css';

    function App() {
      return (
        <div className="App">
          <header className="App-header">
            <img src={logo} className="App-logo" alt="logo" />
            <p>
              Welcome to HNG React!
            </p>
            <a
              className="App-link"
              href="https://reactjs.org"
              target="_blank"
              rel="noopener noreferrer"
            >
              Learn React
            </a>
          </header>
        </div>
      );
    }

    export default App;
    ```

    Save the file and watch your browser automatically refresh with the updated text.

    <Note>
      Hot Module Replacement (HMR) is enabled by default, so most changes will reflect instantly without a full page reload.
    </Note>
  </Step>
</Steps>

## Project Structure

Here's what you'll find in the HNG React project:

```text
hng/
├── public/
│   ├── index.html          # HTML template
│   ├── favicon.ico         # App icon
│   └── manifest.json       # PWA manifest
├── src/
│   ├── App.js              # Main App component
│   ├── App.css             # App styles
│   ├── index.js            # Application entry point
│   ├── index.css           # Global styles
│   └── reportWebVitals.js  # Performance monitoring
└── package.json            # Project dependencies
```

### Key Files

**src/index.js** - Application entry point:
```jsx src/index.js
import React from 'react';
import ReactDOM from 'react-dom/client';
import './index.css';
import App from './App';
import reportWebVitals from './reportWebVitals';

const root = ReactDOM.createRoot(document.getElementById('root'));
root.render(
  <React.StrictMode>
    <App />
  </React.StrictMode>
);

reportWebVitals();
```

**src/App.js** - Main component:
```jsx src/App.js
import logo from './logo.svg';
import './App.css';

function App() {
  return (
    <div className="App">
      <header className="App-header">
        <img src={logo} className="App-logo" alt="logo" />
        <p>
          Edit <code>src/App.js</code> and save to reload.
        </p>
        <a
          className="App-link"
          href="https://reactjs.org"
          target="_blank"
          rel="noopener noreferrer"
        >
          Learn React
        </a>
      </header>
    </div>
  );
}

export default App;
```

## Available Scripts

The project includes several useful commands:

### Development
```bash
npm start
```
Runs the app in development mode with hot reloading at http://localhost:3000.

### Testing
```bash
npm test
```
Launches the test runner in interactive watch mode.

### Production Build
```bash
npm run build
```
Builds the app for production to the `build` folder. The build is minified and optimized for best performance.

<Warning>
  The `npm run eject` command is available but is a one-way operation. Only use it if you need full control over the webpack configuration.
</Warning>
