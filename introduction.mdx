    Quick Start
  </h2>
  <p className="text-base text-gray-600 dark:text-gray-400 mb-8">
    Get your development environment up and running in minutes
  </p>

  <Steps>
    <Step title="Install dependencies">
      Clone the repository and install all required packages using npm.

      ```bash npm
      npm install
      ```

      This installs React 18.2.0, React DOM, testing libraries, and all development dependencies configured in `package.json`.
    </Step>

    <Step title="Start the development server">
      Launch the local development server with hot module replacement.

      ```bash
      npm start
      ```

      <Note>
        The app will automatically open in your browser at [http://localhost:3000](http://localhost:3000). The page reloads whenever you make changes to the source files.
      </Note>
    </Step>

    <Step title="Explore the project structure">
      The project follows the standard Create React App structure with these key directories:

      - **`src/`** - All React components and application code
      - **`public/`** - Static assets and the HTML template
      - **`src/App.js`** - Main application component
      - **`src/index.js`** - React entry point using `createRoot` API

      Learn more in the [Project Structure](/development/project-structure) guide.
    </Step>

    <Step title="Build your first component">
      Start editing `src/App.js` to customize your application. Changes are reflected instantly in the browser.

      ```jsx App.js
      function App() {
        return (
          <div className="App">
            <header className="App-header">
              <h1>Welcome to HNG</h1>
              <p>Start building amazing things!</p>
            </header>
          </div>
        );
      }

      export default App;
      ```
    </Step>
  </Steps>
</div>
